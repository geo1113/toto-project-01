# Blueprint: 꿀팁 저장소

## 프로젝트 개요

일상의 유용한 팁과 정보를 공유하고 소통하는 것을 목표로 하는 블로그 형태의 웹 애플리케이션입니다.

## 스타일 및 디자인 가이드

- **레이아웃:** 반응형 웹 디자인을 적용하여 모바일과 데스크톱 환경 모두에서 최적의 사용자 경험을 제공합니다.
- **색상 팔레트:** 라이트/다크 모드를 지원하여 사용자의 시각적 편안함을 고려합니다.
- **폰트:** 가독성이 높은 시스템 폰트를 사용하여 명확한 정보 전달을 목표로 합니다.
- **상호작용:** 직관적인 LNB 메뉴와 검색 기능을 통해 원하는 정보에 쉽게 접근할 수 있습니다.

## 기능 명세

- **한/영 키보드 변환기:**
  - **UI:** 좌/우 두 개의 텍스트 영역(원본, 결과), 변환 방향 선택 라디오 버튼(한→영, 영→한), 입력/결과 텍스트 교환 버튼, 각 텍스트 영역 하단의 바이트(byte) 표시 공간으로 구성됩니다.
  - **실시간 변환:** 사용자가 원본 텍스트 영역에 키보드로 입력하거나 텍스트를 붙여넣으면, 선택된 변환 방향에 따라 실시간으로 한/영 키보드 입력을 변환하여 결과 텍스트 영역에 보여줍니다.
  - **바이트 계산:** 원본과 결과 텍스트의 바이트 크기를 각각 계산하여 표시합니다.
  - **기술 구현:**
    - HTML: `post1.html`에 변환기 UI 요소를 배치합니다.
    - JavaScript: `js/converter.js`라는 별도 모듈 파일을 생성하여 변환 로직, 이벤트 처리, 바이트 계산 등 모든 동적 기능을 관리합니다.
    - CSS: `style.css`에 변환기 레이아웃, 반응형 디자인, 컨트롤 스타일을 추가하여 기존 블로그 디자인과 일관성을 유지합니다.

- **계층형 게시물 구조 (LNB):**
  - `posts.json`에 `subPosts` 배열을 정의하여 게시물 간의 부모-자식 관계를 설정할 수 있습니다.
  - LNB에서 부모 게시물 옆에 토글 아이콘(▼/▲)을 표시합니다.
  - 사용자가 아이콘을 클릭하면 하위 게시물 목록이 들여쓰기 된 상태로 나타나거나 사라져, 계층 구조를 명확하게 탐색할 수 있습니다.

- **게시물 관리:** `posts.json`을 통해 게시물 목록을 동적으로 관리하며, 개별 HTML 파일을 통해 콘텐츠를 제공합니다.

- **다크 모드:** 사용자가 테마를 전환할 수 있는 버튼을 제공합니다.

- **검색:** LNB 하단의 검색창을 통해 게시물 제목을 기준으로 필터링합니다. 계층 구조를 인식하여 부모 또는 자식 포스트가 검색어와 일치하면 모두 표시합니다.

- **댓글:** Disqus를 연동하여 각 게시물에 대한 사용자 소통을 활성화합니다.

- **다국어 지원:** Google 번역 위젯을 통해 한국어와 영어를 지원합니다.

## 현재 진행 계획: 한/영 키보드 변환기 구현 완료

### 목표

기존 "효과적인 시간 관리 비법" 포스트를 "한/영 키보드 변환기" 기능으로 대체하여 사용자에게 새로운 유틸리티를 제공합니다.

### 실행 완료된 단계

1.  **LNB 메뉴 업데이트:** `posts.json` 파일의 `post1.html`에 연결된 `title`을 "한/영 키보드 변환기"로 수정했습니다.
2.  **HTML 구조 개편:** `posts/post1.html`의 내용을 지우고, 변환기 기능을 위한 새로운 UI(텍스트 영역, 버튼, 정보 표시 공간)를 작성했습니다.
3.  **핵심 로직 구현:** 한/영 변환, 바이트 계산, 이벤트 처리 로직을 담은 `js/converter.js` 파일을 새로 생성하고 구현을 완료했습니다.
4.  **스타일 적용:** `style.css`에 변환기 UI가 기존 디자인과 조화를 이루고 반응형으로 동작하도록 스타일 규칙을 추가했습니다.
5.  **Blueprint 업데이트:** 본 `blueprint.md` 파일을 수정하여 새로운 기능 명세와 현재까지의 작업 내역을 정확하게 반영했습니다.
