# 프로젝트 블루프린트: 생활 꿀팁 블로그

## 1. 개요

이 프로젝트는 다양한 생활 꿀팁을 공유하고 구글 애드센스를 통해 수익을 창출하는 것을 목표로 하는 웹사이트입니다. SEO(검색 엔진 최적화)를 고려하여 각 꿀팁은 개별 HTML 파일로 관리되며, 메인 페이지에서는 항상 최신 꿀팁이 노출됩니다. 사용자는 왼쪽 네비게이션 바(LNB)를 통해 모든 꿀팁 목록을 쉽게 탐색할 수 있습니다.

## 2. 디자인 및 기능 명세

### 스타일 및 디자인 (V4)

- **레이아웃:** 2단 레이아웃 (LNB + 본문)
- **테마:** 라이트 모드와 다크 모드를 지원합니다. 사용자의 설정은 로컬 스토리지에 저장되어 유지됩니다.
- **디자인 컨셉:** CSS 변수를 활용한 테마 시스템을 구축하여 일관된 색상 팔레트를 유지합니다.
- **반응형:** 데스크톱과 모바일 환경 모두에서 최적화된 화면을 제공합니다.

### 기능 명세 (V4)

- **설정 툴바 (Web Component):**
    - 우측 상단에 고정된 `<settings-toolbar>` 컴포넌트를 제공합니다.
    - **다크 모드 토글:** 클릭 시 라이트/다크 모드로 전환하며 배경색, 글자색 등을 업데이트합니다.
    - **번역 기능:** 구글 번역 API와 연동하여 한국어/영어 간의 번역 기능을 제공합니다. 버튼 클릭 시 `lang` 속성을 변경하고 번역 엔진을 실행합니다.
- **동적 콘텐츠 로딩:**
    - `posts.json` 파일을 데이터베이스로 사용하여 게시물 목록을 관리합니다.
- **LNB 검색 기능:**
    - 실시간 제목 필터링 기능을 제공합니다.
- **SEO 최적화:**
    - 각 게시물은 개별 HTML 파일로 관리됩니다.

### 코드 구조 (V4)

- **JavaScript 모듈화:** 
    - `js/api.js`: 데이터 관리.
    - `js/ui.js`: UI 렌더링 및 **SettingsToolbar 웹 컴포넌트** 정의.
    - `js/router.js`: 라우팅 및 히스토리 관리.
    - `main.js`: 애플리케이션 진입점 및 초기화.

## 3. 현재 구현 계획

1.  **CSS 변수 및 테마 스타일 추가:** `style.css`에 라이트/다크 모드 전용 변수 및 설정 버튼 스타일을 추가했습니다. (완료)
2.  **`SettingsToolbar` 컴포넌트 개발:** `js/ui.js`에 다크모드 및 번역 로직을 포함한 웹 컴포넌트를 정의했습니다. (완료)
3.  **번역 위젯 연동:** 구글 번역 스크립트를 동적으로 로드하고 버튼 클릭 시 제어할 수 있는 로직을 구현했습니다. (완료)
4.  **`index.html` 적용:** 메인 레이아웃에 `<settings-toolbar>`를 삽입했습니다. (완료)
