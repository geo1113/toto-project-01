# 프로젝트 블루프린트: 생활 꿀팁 블로그

## 1. 개요

이 프로젝트는 다양한 생활 꿀팁을 공유하고 구글 애드센스를 통해 수익을 창출하는 것을 목표로 하는 웹사이트입니다. SEO(검색 엔진 최적화)를 고려하여 각 꿀팁은 개별 HTML 파일로 관리되며, 메인 페이지에서는 항상 최신 꿀팁이 노출됩니다. 사용자는 왼쪽 네비게이션 바(LNB)를 통해 모든 꿀팁 목록을 쉽게 탐색할 수 있습니다.

## 2. 디자인 및 기능 명세

### 스타일 및 디자인 (V3)

- **레이아웃:** 2단 레이아웃 (LNB + 본문)
- **디자인 컨셉:** 깔끔하고 현대적인 디자인을 적용하여 가독성을 높입니다.
- **반응형:** 데스크톱과 모바일 환경 모두에서 최적화된 화면을 제공합니다.

### 기능 명세 (V3)

- **동적 콘텐츠 로딩:**
    - `posts.json` 파일을 "데이터베이스"로 사용하여 게시물 목록과 메타데이터(제목, 파일명, 작성일)를 관리합니다.
- **LNB 검색 기능:**
    - LNB 상단에 위치한 검색창에 사용자가 키워드를 입력하면, 게시물 제목을 기준으로 실시간으로 목록을 필터링하여 보여줍니다.
- **SEO 최적화:**
    - 각 꿀팁은 고유한 URL을 가진 개별 `.html` 파일로 존재하여 검색 엔진이 콘텐츠를 쉽게 수집하고 인덱싱할 수 있습니다.
- **수익 창출:**
    - 구글 애드센스 스크립트를 삽입할 수 있는 공간을 마련합니다.

### 코드 구조 (V3)

- **JavaScript 모듈화:** 코드의 유지보수성과 가독성을 높이기 위해 기능별로 JS 파일을 분리합니다.
    - `js/api.js`: 데이터(`posts.json`)를 가져오고 처리하는 역할.
    - `js/ui.js`: DOM 조작(UI 렌더링, 콘텐츠 삽입 등)을 담당하는 역할.
    - `js/router.js`: URL 변경, 히스토리 관리 등 라우팅 로직을 담당하는 역할.
    - `main.js`: 각 모듈을 가져와 애플리케이션을 초기화하고 전체 흐름을 제어하는 진입점 역할.

## 3. 현재 구현 계획

1.  **`blueprint.md` 업데이트:** JavaScript 모듈화 리팩토링 계획을 문서에 반영합니다. (완료)
2.  **`js` 디렉터리 생성:** 모듈 파일들을 저장할 폴더를 생성합니다.
3.  **`js/api.js` 생성:** 데이터 fetching 및 처리 관련 함수를 작성합니다.
4.  **`js/ui.js` 생성:** UI 렌더링 및 조작 관련 함수들을 작성합니다.
5.  **`js/router.js` 생성:** 라우팅 관련 함수들을 작성합니다.
6.  **`main.js` 리팩토링:** 기존 코드를 제거하고, 각 모듈을 `import`하여 애플리케이션을 조립하는 코드로 재작성합니다.
7.  **`index.html` 수정:** `<script>` 태그에 `type="module"` 속성을 추가하여 모듈 시스템을 활성화합니다.
