# Blueprint: 꿀팁 저장소

## 프로젝트 개요

일상의 유용한 팁과 정보를 공유하고 소통하는 것을 목표로 하는 블로그 형태의 웹 애플리케이션입니다.

## 스타일 및 디자인 가이드

- **레이아웃:** 반응형 웹 디자인을 적용하여 모바일과 데스크톱 환경 모두에서 최적의 사용자 경험을 제공합니다.
- **색상 팔레트:** 라이트/다크 모드를 지원하여 사용자의 시각적 편안함을 고려합니다.
- **폰트:** 가독성이 높은 시스템 폰트를 사용하여 명확한 정보 전달을 목표로 합니다.
- **상호작용:** 직관적인 LNB 메뉴와 검색 기능을 통해 원하는 정보에 쉽게 접근할 수 있습니다.

## 기능 명세

- **계층형 게시물 구조 (LNB):**
  - `posts.json`에 `subPosts` 배열을 정의하여 게시물 간의 부모-자식 관계를 설정할 수 있습니다.
  - LNB에서 부모 게시물 옆에 토글 아이콘(▼/▲)을 표시합니다.
  - 사용자가 아이콘을 클릭하면 하위 게시물 목록이 들여쓰기 된 상태로 나타나거나 사라져, 계층 구조를 명확하게 탐색할 수 있습니다.
- **게시물 관리:** `posts.json`을 통해 게시물 목록을 동적으로 관리하며, 개별 HTML 파일을 통해 콘텐츠를 제공합니다.
- **다크 모드:** 사용자가 테마를 전환할 수 있는 버튼을 제공합니다.
- **검색:** LNB 하단의 검색창을 통해 게시물 제목을 기준으로 필터링합니다. 계층 구조를 인식하여 부모 또는 자식 포스트가 검색어와 일치하면 모두 표시합니다.
- **댓글:** Disqus를 연동하여 각 게시물에 대한 사용자 소통을 활성화합니다.
- **다국어 지원:** Google 번역 위젯을 통해 한국어와 영어를 지원합니다.

## 현재 진행 계획: Google AdSense 연동

### 목표

Google AdSense를 웹사이트에 연동하여 광고 수익을 창출합니다.

### 실행 단계

1.  **AdSense 스크립트 삽입:**
    -   사용자로부터 AdSense 클라이언트 ID를 전달받습니다.
    -   `index.html`의 `<head>` 태그에 있는 플레이스홀더를 실제 클라이언트 ID로 교체합니다.

2.  **광고 단위 배치:**
    -   **LNB 상단:** 왼쪽 내비게이션 바의 최상단에 디스플레이 광고를 추가합니다.
    -   **메인 콘텐츠 상단:** `main-content` 영역의 시작 부분에 디스플레이 광고를 추가하여 사용자의 시선이 처음 머무는 곳에 광고를 노출합니다.
    -   **게시물 본문 상단:** 각 포스트(`post1.html`, `post2.html` 등)의 `<article>` 태그 내 시작 부분에 광고를 추가하여 콘텐츠 소비와 함께 자연스럽게 광고를 노출시킵니다.
    -   각 광고 단위에는 AdSense에서 제공하는 광고 코드를 삽입할 수 있도록 주석으로 영역을 표시합니다.

3.  **최종 확인 및 커밋:**
    -   광고가 정상적으로 표시될 영역이 확보되었는지 확인합니다.
    -   관련 변경사항을 GitHub에 커밋하여 작업을 완료합니다.
