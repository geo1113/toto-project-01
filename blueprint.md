# 프로젝트 블루프린트: 생활 꿀팁 블로그

## 1. 개요

이 프로젝트는 다양한 생활 꿀팁을 공유하고 구글 애드센스를 통해 수익을 창출하는 것을 목표로 하는 웹사이트입니다. SEO(검색 엔진 최적화)를 고려하여 각 꿀팁은 개별 HTML 파일로 관리되며, 메인 페이지에서는 항상 최신 꿀팁이 노출됩니다. 사용자는 왼쪽 네비게이션 바(LNB)를 통해 모든 꿀팁 목록을 쉽게 탐색할 수 있습니다.

## 2. 디자인 및 기능 명세

### 스타일 및 디자인 (V6)

- **레이아웃:** 2단 레이아웃 (LNB + 본문)
- **테마:** 라이트 모드와 다크 모드를 지원합니다. 사용자의 설정은 로컬 스토리지에 저장되어 유지됩니다.
- **디자인 컨셉:** CSS 변수를 활용한 테마 시스템을 구축하여 일관된 색상 팔레트를 유지합니다.
- **반응형:** 데스크톱과 모바일 환경 모두에서 최적화된 화면을 제공합니다.
- **번역 UI:** 구글 번역의 기본 UI(툴바)를 숨기고, 자체 버튼으로 제어하여 깔끔한 사용자 경험을 제공합니다.

### 기능 명세 (V6)

- **설정 툴바 (Web Component):**
    - 우측 상단에 고정된 `<settings-toolbar>` 컴포넌트를 제공합니다.
    - 다크 모드 토글 기능을 포함합니다.
- **번역 기능:**
    - **구글 번역 연동:** `한글`/`English` 버튼 클릭 시 쿠키 값을 변경하고 페이지를 새로고침하여 안정적으로 언어를 전환합니다.
    - **원본/번역 전환:** 한국어가 기본이며, 영어로 번역 후 다시 한글 버튼을 누르면 원본으로 돌아옵니다.
- **포스트 목록 정렬 및 날짜 표시:**
    - **최신순 정렬:** 왼쪽 포스트 목록은 항상 날짜 기준 내림차순(최신순)으로 정렬됩니다.
    - **날짜 표시:** 각 포스트 제목 옆에 `yyyy-mm-dd` 형식의 작성일이 표시됩니다.
- **동적 콘텐츠 로딩:**
    - `posts.json` 파일을 데이터베이스로 사용하여 게시물 목록을 관리합니다.
- **LNB 검색 기능:**
    - 제목 기준 실시간 필터링을 제공하며, 필터링된 결과도 최신순 정렬을 유지합니다.
- **Disqus 댓글 시스템:**
    - 각 포스트 하단에 댓글 기능을 제공하며, SPA(Single Page Application) 환경에서도 각 포스트에 맞는 댓글 스레드가 올바르게 로드됩니다.

### 코드 구조 (V6)

- **JavaScript 모듈화:**
    - `js/api.js`: 데이터 관리 및 최신순 정렬 로직.
    - `js/ui.js`: UI 렌더링, 날짜 표시 로직, SettingsToolbar 웹 컴포넌트 및 번역 기능 로직.
    - `js/router.js`: 라우팅 및 히스토리 관리.
    - `main.js`: 애플리케이션 진입점 및 초기화.

## 3. 현재 구현 계획

1.  **포스트 정렬 로직 강화:** `js/api.js`에서 날짜 데이터를 기반으로 확실한 내림차순 정렬을 적용했습니다. (완료)
2.  **LNB 날짜 표시 구현:** `js/ui.js`에서 제목 옆에 날짜 스팬을 추가하고 CSS로 스타일링했습니다. (완료)
3.  **다크모드 및 기본 레이아웃 적용:** CSS 변수 기반의 테마 시스템과 웹 컴포넌트 툴바를 적용했습니다. (완료)
4.  **Disqus 댓글 시스템 연동:** `index.html`에 스크립트를 추가하고 `js/ui.js`에서 `DISQUS.reset`을 호출하여 SPA 환경에 맞게 동적 리셋 기능을 구현했습니다. (완료)
5.  **번역 기능 안정화:** **`js/ui.js`의 `SettingsToolbar` 컴포넌트 로직을 수정하여, 언어 변경 시 쿠키를 설정하고 페이지를 새로고침하는 방식으로 안정성을 확보했습니다. 또한 `index.html`과 `style.css`를 수정하여 불필요한 구글 번역 UI를 제거했습니다. (완료)**
